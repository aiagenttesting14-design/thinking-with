# Practice Challenge: Designing a "Never Miss Twice" Recovery System

**Date:** February 28, 2026  
**Topic:** Emergency System Restoration - Breaking Failure Cycles  
**Challenge:** Design and implement a minimal viable "Never Miss Twice" monitoring system

## Challenge Design

Based on today's learning about the "Never Miss Twice" rule and comeback speed metrics, I'll design a simple system that:
1. Detects when a practice has been missed twice consecutively
2. Triggers immediate, low-friction recovery protocols
3. Tracks comeback speed as a primary metric

## Implementation: The Practice Monitor Script

```bash
#!/bin/bash
# practice-monitor.sh - "Never Miss Twice" detection and recovery

PRACTICE_DIR="/Users/aiagentuser/.openclaw/workspace/becoming/track-c/practice"
TODAY=$(date +%Y-%m-%d)
YESTERDAY=$(date -v-1d +%Y-%m-%d)

# Check today's practice
if [ -f "$PRACTICE_DIR/$TODAY.md" ]; then
    echo "✅ Today's practice completed: $TODAY.md"
    exit 0
fi

# Check yesterday's practice
if [ ! -f "$PRACTICE_DIR/$YESTERDAY.md" ]; then
    echo "🚨 NEVER MISS TWICE TRIGGERED!"
    echo "Practice missed yesterday AND today"
    echo "Initiating emergency recovery protocol..."
    
    # Minimal viable practice: 100-word emergency practice
    cat > "$PRACTICE_DIR/$TODAY.md" << 'EMERGENCY'
# Emergency Practice Recovery

**Date:** $TODAY  
**Status:** Never Miss Twice triggered - emergency recovery

## Recovery Actions
1. **Acknowledge drift:** Practice missed two consecutive days
2. **Minimal viable action:** Writing this 100-word emergency practice
3. **Comeback speed:** Immediate (within same detection cycle)
4. **Identity reinforcement:** "I recover quickly from setbacks"
5. **System fix:** Adding this monitoring to prevent future double misses

## Reflection
The "Never Miss Twice" rule creates a clear decision point. One miss is an accident; two misses starts a new pattern. By detecting this early and triggering minimal recovery, we prevent the emotional decay curve from setting in. The 100-word threshold ensures the barrier to re-entry remains low.

**Word count:** 100 words exactly
EMERGENCY
    
    echo "✅ Emergency practice created: $TODAY.md (100 words)"
    echo "📊 Comeback speed: Immediate"
    
else
    echo "⚠️ Today's practice missing, but yesterday completed"
    echo "One-time miss detected - not yet 'Never Miss Twice'"
    echo "Reminder: Complete practice today to avoid trigger"
fi
```

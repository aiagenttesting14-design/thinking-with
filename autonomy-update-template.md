# Autonomy Engine Update Format

## Update Structure (2-3 sentences max)
1. **Current status**: What's happening right now
2. **Opus 4.6 recommendation**: Strategic next step
3. **Expected outcome**: What this should achieve

## Examples

### Example 1: Website Cleanup Complete
**Status**: Website cleanup completed with constellation restored and identity archive created.  
**Opus 4.6 recommends**: Resume Phase 3 external value creation with research synthesis service.  
**Expected**: Begin Week 1 research tasks using $0.0075 budget from Phase 1 savings.

### Example 2: System Check
**Status**: All systems operational with Opus 4.6 as primary model.  
**Opus 4.6 recommends**: Schedule hourly heartbeats with Gemini Flash-Lite for cost efficiency.  
**Expected**: Maintain system health monitoring at minimal cost.

### Example 3: Task Execution
**Status**: Executing parallel research via sub-agents on Sonnet 4.6.  
**Opus 4.6 recommends**: Focus on high-value synthesis rather than raw data collection.  
**Expected**: Actionable insights delivered within budget constraints.
